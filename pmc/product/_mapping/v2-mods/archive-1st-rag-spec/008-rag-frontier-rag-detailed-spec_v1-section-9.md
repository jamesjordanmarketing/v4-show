## SECTION 9: Quality Measurement System

**Extension Status**: NEW -- all service functions, evaluation prompts, and aggregation logic are new additions. Follows the existing `quality-feedback-service.ts` pattern for aggregation and the `claude-api-client.ts` pattern for LLM calls.

---

### Overview

This section implements the **Claude-as-Judge** quality measurement system for the Frontier RAG module. Every response generated by the RAG chat system is automatically evaluated against 5 metrics by Claude Haiku, producing a composite quality score stored in the `rag_quality_scores` table. The system provides aggregated metrics, trend data, query history, and the critical **mode comparison** feature that demonstrates RAG+LoRA > RAG alone > LoRA alone.

**User value delivered:**
- Automated, consistent quality evaluation on every response
- Quantitative evidence that the RAG system works (for investors and users)
- Side-by-side comparison of three modes (the "hero feature")
- Historical trend tracking to see improvement over time
- Per-query drill-down for debugging retrieval quality

**What already exists (reused):**
- Supabase client (`import { supabase } from '@/lib/supabase'`) for database queries
- `rag_quality_scores` and `rag_queries` tables (Section 1)

- Claude Haiku model for evaluation (already available via Anthropic SDK)
- Error handling patterns (`try/catch`, `console.error`, throw)

### Features & Requirements

#### FR-9.1: RAG Quality Evaluation Service

**Type**: Service

**Description**: The core evaluation function that takes a completed query-response pair and sends it to Claude Haiku for quality assessment across 5 metrics. The LLM returns structured JSON scores, which are validated, weighted into a composite score, and stored in the `rag_quality_scores` table. This function is called automatically after every RAG response is generated (from the retrieval pipeline in Section 5, invoked via the query API route in Section 6).

**Implementation Strategy**: NEW build

**File**: `src/lib/services/rag/rag-quality-service.ts`

```typescript
/**
 * RAG Quality Evaluation Service
 *
 * Implements Claude-as-Judge quality measurement for every RAG response.
 * Evaluates 5 metrics (faithfulness, answer relevance, context relevance,
 * answer completeness, citation accuracy), computes a weighted composite
 * score, and stores results in rag_quality_scores.
 *
 * Pattern Source: quality-feedback-service.ts (aggregation), claude-api-client.ts (LLM calls)
 *
 * @module rag-quality-service
 */

import { supabase } from '@/lib/supabase';
import { getLLMProvider } from '@/lib/providers/llm-provider';
import type {
  QualityEvaluation,
  QualityMetrics,
  QualityScoreRow,
  QueryHistoryEntry,
  ModeComparison,
  ModeBreakdown,
  RAGMode,
  Citation,
} from '@/types/rag';

// ============================================================================
// Constants
// ============================================================================

/**
 * Metric weights for composite score calculation.
 * Sum = 1.00. Faithfulness is highest because hallucination
 * is the primary failure mode in RAG systems.
 */
const METRIC_WEIGHTS = {
  faithfulness: 0.30,
  answerRelevance: 0.25,
  contextRelevance: 0.20,
  answerCompleteness: 0.15,
  citationAccuracy: 0.10,
} as const;

/**
 * The structured evaluation prompt sent to Claude Haiku.
 * Instructs the model to return a JSON object with scores and reasoning.
 *
 * Design rationale:
 * - Each metric has a clear definition and scoring rubric
 * - Reasoning fields force the model to "think before scoring"
 * - JSON output is parseable without regex extraction
 * - 0-1 scale matches the NUMERIC(3,2) column constraints
 */
const EVALUATION_SYSTEM_PROMPT = `You are an expert evaluator for a Retrieval-Augmented Generation (RAG) system. Your job is to assess the quality of a generated response given the user's query, the retrieved context, and any citations provided.

You must evaluate the response on exactly 5 metrics. For each metric, provide:
1. A score from 0.00 to 1.00 (two decimal places)
2. A brief reasoning (1-2 sentences) explaining your score

## Metrics

### 1. Faithfulness (0.00 - 1.00)
Does every claim in the response have supporting evidence in the provided context?
- 1.00: Every claim is directly grounded in the context. No fabricated information.
- 0.75: Almost all claims are grounded. Minor inferences that are reasonable.
- 0.50: Some claims are grounded, but there are unsupported statements.
- 0.25: Many claims lack support in the context.
- 0.00: The response fabricates information not in the context.

### 2. Answer Relevance (0.00 - 1.00)
Does the response directly and completely answer the user's question?
- 1.00: Directly answers the question with appropriate detail.
- 0.75: Answers the question but includes some tangential information.
- 0.50: Partially answers the question; misses key aspects.
- 0.25: Mostly irrelevant to the question asked.
- 0.00: Does not address the question at all.

### 3. Context Relevance (0.00 - 1.00)
Was the retrieved context relevant to answering the question?
- 1.00: All retrieved context is highly relevant to the question.
- 0.75: Most context is relevant; some noise but not distracting.
- 0.50: Mixed relevance; about half the context is useful.
- 0.25: Mostly irrelevant context was retrieved.
- 0.00: No relevant context was retrieved.

### 4. Answer Completeness (0.00 - 1.00)
Does the response capture all key information available in the context that is relevant to the question?
- 1.00: All relevant information from the context is included.
- 0.75: Most relevant information is included; minor omissions.
- 0.50: Key information is missing but the answer is partially useful.
- 0.25: Major relevant information is omitted.
- 0.00: The response is essentially empty or captures nothing useful.

### 5. Citation Accuracy (0.00 - 1.00)
Are the citations correct? Do they point to the right source sections/facts?
- 1.00: All citations are accurate and point to the correct sources.
- 0.75: Most citations are correct; minor attribution errors.
- 0.50: Some citations are correct, others are misattributed.
- 0.25: Most citations are incorrect or missing.
- 0.00: No citations provided, or all citations are wrong.
- If no citations were expected (e.g., simple factual answer), score 1.00.

## Output Format

You MUST respond with ONLY a valid JSON object in this exact structure. No markdown, no explanation outside the JSON:

{
  "faithfulness": { "score": 0.00, "reasoning": "..." },
  "answer_relevance": { "score": 0.00, "reasoning": "..." },
  "context_relevance": { "score": 0.00, "reasoning": "..." },
  "answer_completeness": { "score": 0.00, "reasoning": "..." },
  "citation_accuracy": { "score": 0.00, "reasoning": "..." }
}`;

// ============================================================================
// FR-9.1: Core Evaluation Function
// ============================================================================

/**
 * Evaluate the quality of a RAG response using Claude-as-Judge.
 *
 * Sends the query, assembled context, response, and citations to Claude Haiku,
 * which returns structured scores for 5 metrics. Computes the composite score
 * and stores the result in rag_quality_scores.
 *
 * @param params - The query ID, query text, assembled context, response text, and citations
 * @returns The complete quality evaluation with all scores and metadata
 *
 * @example
 * ```typescript
 * const evaluation = await evaluateResponseQuality({
 *   queryId: 'abc-123',
 *   queryText: 'What are the treatment protocols?',
 *   assembledContext: '...retrieved sections and facts...',
 *   responseText: 'The treatment protocols include...',
 *   citations: [{ sectionId: '...', text: '...' }],
 *   userId: 'user-456',
 * });
 * console.log(evaluation.compositeScore); // 0.82
 * ```
 */
export async function evaluateResponseQuality(params: {
  queryId: string;
  queryText: string;
  assembledContext: string;
  responseText: string;
  citations: Citation[] | null;
  userId: string;
}): Promise<QualityEvaluation> {
  const { queryId, queryText, assembledContext, responseText, citations, userId } = params;

  const startTime = Date.now();

  try {
    // Build the user message for evaluation
    const userMessage = buildEvaluationUserMessage(queryText, assembledContext, responseText, citations);

    // Call LLM provider's evaluateQuality method (routes to Claude Haiku)
    const llmProvider = getLLMProvider();
    const evaluationResult = await llmProvider.evaluateQuality({
      systemPrompt: EVALUATION_SYSTEM_PROMPT,
      userMessage,
    });

    // Parse the structured JSON response
    const parsedScores = parseEvaluationResponse(evaluationResult.content);

    // Compute composite score
    const compositeScore = computeCompositeScore({
      faithfulness: parsedScores.faithfulness.score,
      answerRelevance: parsedScores.answer_relevance.score,
      contextRelevance: parsedScores.context_relevance.score,
      answerCompleteness: parsedScores.answer_completeness.score,
      citationAccuracy: parsedScores.citation_accuracy.score,
    });

    const evaluationTimeMs = Date.now() - startTime;

    // Store in rag_quality_scores table
    const { data: storedScore, error: insertError } = await supabase
      .from('rag_quality_scores')
      .insert({
        query_id: queryId,
        user_id: userId,
        faithfulness: parsedScores.faithfulness.score,
        answer_relevance: parsedScores.answer_relevance.score,
        context_relevance: parsedScores.context_relevance.score,
        answer_completeness: parsedScores.answer_completeness.score,
        citation_accuracy: parsedScores.citation_accuracy.score,
        composite_score: compositeScore,
        evaluation_model: evaluationResult.model || 'claude-haiku-4-5-20250929',
        evaluation_tokens_used: evaluationResult.tokensUsed || null,
        evaluation_time_ms: evaluationTimeMs,
        raw_evaluation: parsedScores,
      })
      .select()
      .single();

    if (insertError) {
      console.error('Error storing quality score:', insertError);
      throw new Error(`Failed to store quality score: ${insertError.message}`);
    }

    // Return the complete evaluation
    const evaluation: QualityEvaluation = {
      id: storedScore.id,
      queryId,
      faithfulness: parsedScores.faithfulness.score,
      answerRelevance: parsedScores.answer_relevance.score,
      contextRelevance: parsedScores.context_relevance.score,
      answerCompleteness: parsedScores.answer_completeness.score,
      citationAccuracy: parsedScores.citation_accuracy.score,
      compositeScore,
      evaluationModel: evaluationResult.model || 'claude-haiku-4-5-20250929',
      evaluationTokensUsed: evaluationResult.tokensUsed || null,
      evaluationTimeMs,
      reasoning: {
        faithfulness: parsedScores.faithfulness.reasoning,
        answerRelevance: parsedScores.answer_relevance.reasoning,
        contextRelevance: parsedScores.context_relevance.reasoning,
        answerCompleteness: parsedScores.answer_completeness.reasoning,
        citationAccuracy: parsedScores.citation_accuracy.reasoning,
      },
      createdAt: storedScore.created_at,
    };

    return evaluation;
  } catch (error) {
    console.error('Error in evaluateResponseQuality:', error);

    // On evaluation failure, store a fallback record so we know evaluation was attempted
    // but do not block the response from being returned to the user
    try {
      await supabase
        .from('rag_quality_scores')
        .insert({
          query_id: queryId,
          user_id: userId,
          faithfulness: 0,
          answer_relevance: 0,
          context_relevance: 0,
          answer_completeness: 0,
          citation_accuracy: 0,
          composite_score: 0,
          evaluation_model: 'error-fallback',
          evaluation_time_ms: Date.now() - startTime,
          raw_evaluation: {
            error: error instanceof Error ? error.message : 'Unknown evaluation error',
            timestamp: new Date().toISOString(),
          },
        });
    } catch (fallbackError) {
      console.error('Error storing fallback quality score:', fallbackError);
    }

    throw error;
  }
}

/**
 * Build the user message for the evaluation prompt.
 * Formats the query, context, response, and citations into a structured message
 * that Claude can evaluate.
 */
function buildEvaluationUserMessage(
  queryText: string,
  assembledContext: string,
  responseText: string,
  citations: Citation[] | null
): string {
  const citationText = citations && citations.length > 0
    ? citations.map((c, i) => `  [${i + 1}] Section: ${c.sectionId || 'N/A'}, Fact: ${c.factId || 'N/A'}, Text: "${c.textSnippet}"`).join('\n')
    : '  (No citations provided)';

  return `## User Query
${queryText}

## Retrieved Context
${assembledContext}

## Generated Response
${responseText}

## Citations
${citationText}

Please evaluate the response quality and return your assessment as JSON.`;
}

/**
 * Parse and validate the structured JSON response from Claude.
 * Handles edge cases: markdown code fences, trailing text, invalid JSON.
 */
function parseEvaluationResponse(content: string): {
  faithfulness: { score: number; reasoning: string };
  answer_relevance: { score: number; reasoning: string };
  context_relevance: { score: number; reasoning: string };
  answer_completeness: { score: number; reasoning: string };
  citation_accuracy: { score: number; reasoning: string };
} {
  // Strip markdown code fences if present
  let cleaned = content.trim();
  if (cleaned.startsWith('```json')) {
    cleaned = cleaned.slice(7);
  } else if (cleaned.startsWith('```')) {
    cleaned = cleaned.slice(3);
  }
  if (cleaned.endsWith('```')) {
    cleaned = cleaned.slice(0, -3);
  }
  cleaned = cleaned.trim();

  try {
    const parsed = JSON.parse(cleaned);

    // Validate all 5 metrics exist and have valid scores
    const metrics = ['faithfulness', 'answer_relevance', 'context_relevance', 'answer_completeness', 'citation_accuracy'];
    for (const metric of metrics) {
      if (!parsed[metric] || typeof parsed[metric].score !== 'number') {
        throw new Error(`Missing or invalid metric: ${metric}`);
      }
      // Clamp scores to valid range
      parsed[metric].score = Math.max(0, Math.min(1, Number(parsed[metric].score.toFixed(2))));
      // Default reasoning if missing
      if (!parsed[metric].reasoning || typeof parsed[metric].reasoning !== 'string') {
        parsed[metric].reasoning = 'No reasoning provided';
      }
    }

    return parsed;
  } catch (parseError) {
    console.error('Failed to parse evaluation response:', parseError);
    console.error('Raw content:', content);

    // Return neutral scores on parse failure rather than crashing
    return {
      faithfulness: { score: 0.50, reasoning: 'Evaluation parse error - neutral score assigned' },
      answer_relevance: { score: 0.50, reasoning: 'Evaluation parse error - neutral score assigned' },
      context_relevance: { score: 0.50, reasoning: 'Evaluation parse error - neutral score assigned' },
      answer_completeness: { score: 0.50, reasoning: 'Evaluation parse error - neutral score assigned' },
      citation_accuracy: { score: 0.50, reasoning: 'Evaluation parse error - neutral score assigned' },
    };
  }
}

// FR-9.2: Quality Scoring Algorithm
// ============================================================================

/**
 * Compute the weighted composite score from individual metric scores.
 *
 * Weights:
 *   Faithfulness:        0.30 (highest -- hallucination is the primary RAG failure mode)
 *   Answer Relevance:    0.25
 *   Context Relevance:   0.20
 *   Answer Completeness: 0.15
 *   Citation Accuracy:   0.10 (lowest -- citations are a presentation concern)
 *
 * @param scores - Individual metric scores, each 0-1
 * @returns Composite score 0-1 (two decimal places)
 *
 * @example
 * ```typescript
 * const composite = computeCompositeScore({
 *   faithfulness: 0.90,
 *   answerRelevance: 0.85,
 *   contextRelevance: 0.80,
 *   answerCompleteness: 0.75,
 *   citationAccuracy: 0.95,
 * });
 * // composite = 0.30*0.90 + 0.25*0.85 + 0.20*0.80 + 0.15*0.75 + 0.10*0.95
 * //           = 0.270 + 0.2125 + 0.160 + 0.1125 + 0.095
 * //           = 0.85
 * ```
 */
export function computeCompositeScore(scores: {
  faithfulness: number;
  answerRelevance: number;
  contextRelevance: number;
  answerCompleteness: number;
  citationAccuracy: number;
}): number {
  const raw =
    METRIC_WEIGHTS.faithfulness * scores.faithfulness +
    METRIC_WEIGHTS.answerRelevance * scores.answerRelevance +
    METRIC_WEIGHTS.contextRelevance * scores.contextRelevance +
    METRIC_WEIGHTS.answerCompleteness * scores.answerCompleteness +
    METRIC_WEIGHTS.citationAccuracy * scores.citationAccuracy;

  // Round to 2 decimal places and clamp to [0, 1]
  return Math.max(0, Math.min(1, Number(raw.toFixed(2))));
}

/**
 * Get aggregated quality metrics for the quality dashboard.
 *
 * Queries rag_quality_scores joined with rag_queries to compute:
 * - Average composite score and per-metric averages
 * - Per-mode breakdown (rag_only, lora_only, rag_lora)
 * - Total query count
 * - Daily trend data for charting
 *
 * @param params - Filter parameters (userId required, others optional)
 * @returns Structured metrics object for the quality dashboard
 *
 * @example
 * ```typescript
 * const metrics = await getAggregatedQuality({
 *   userId: 'user-123',
 *   knowledgeBaseId: 'kb-456',
 *   fromDate: '2026-02-01',
 *   toDate: '2026-02-09',
 * });
 * console.log(metrics.avgComposite);    // 0.82
 * console.log(metrics.queryCount);      // 47
 * console.log(metrics.modeBreakdown);   // { rag_only: {...}, lora_only: {...}, rag_lora: {...} }
 * ```
 */
export async function getAggregatedQuality(params: {
  userId: string;
  knowledgeBaseId?: string;
  mode?: RAGMode;
  fromDate?: string;
  toDate?: string;
}): Promise<QualityMetrics> {
  const { userId, knowledgeBaseId, mode, fromDate, toDate } = params;

  try {
    // Build query: join rag_quality_scores with rag_queries to get mode and knowledge_base_id
    let query = supabase
      .from('rag_quality_scores')
      .select(`
        id,
        faithfulness,
        answer_relevance,
        context_relevance,
        answer_completeness,
        citation_accuracy,
        composite_score,
        evaluation_model,
        created_at,
        query_id,
        rag_queries!inner (
          id,
          mode,
          knowledge_base_id,
          created_at
        )
      `)
      .eq('user_id', userId)
      .neq('evaluation_model', 'error-fallback'); // Exclude failed evaluations

    // Apply optional filters
    if (knowledgeBaseId) {
      query = query.eq('rag_queries.knowledge_base_id', knowledgeBaseId);
    }
    if (mode) {
      query = query.eq('rag_queries.mode', mode);
    }
    if (fromDate) {
      query = query.gte('created_at', fromDate);
    }
    if (toDate) {
      query = query.lte('created_at', toDate);
    }

    const { data: scores, error } = await query.order('created_at', { ascending: true });

    if (error) {
      console.error('Error fetching aggregated quality:', error);
      throw new Error(`Failed to fetch quality metrics: ${error.message}`);
    }

    if (!scores || scores.length === 0) {
      return buildEmptyMetrics();
    }

    // Compute overall averages
    const queryCount = scores.length;
    const avgComposite = average(scores.map(s => Number(s.composite_score)));
    const avgFaithfulness = average(scores.map(s => Number(s.faithfulness)));
    const avgAnswerRelevance = average(scores.map(s => Number(s.answer_relevance)));
    const avgContextRelevance = average(scores.map(s => Number(s.context_relevance)));
    const avgAnswerCompleteness = average(scores.map(s => Number(s.answer_completeness)));
    const avgCitationAccuracy = average(scores.map(s => Number(s.citation_accuracy)));

    // Compute per-mode breakdown
    const modeBreakdown = computeModeBreakdown(scores);

    // Compute daily trend data for charts
    const trendData = computeTrendData(scores);

    const metrics: QualityMetrics = {
      avgComposite,
      avgFaithfulness,
      avgAnswerRelevance,
      avgContextRelevance,
      avgAnswerCompleteness,
      avgCitationAccuracy,
      queryCount,
      modeBreakdown,
      trendData,
    };

    return metrics;
  } catch (error) {
    console.error('Error in getAggregatedQuality:', error);
    throw error;
  }
}

/**
 * Get query history with quality scores for the query history table.
 *
 * Fetches recent queries joined with their quality scores, sorted by
 * most recent first. Used by the quality dashboard's history table.
 *
 * @param params - Filter parameters (userId required, others optional)
 * @returns Array of query history entries with scores
 *
 * @example
 * ```typescript
 * const history = await getQueryHistory({ userId: 'user-123', limit: 20 });
 * history.forEach(entry => {
 *   console.log(`${entry.queryText} => ${entry.compositeScore}`);
 * });
 * ```
 */
export async function getQueryHistory(params: {
  userId: string;
  knowledgeBaseId?: string;
  limit?: number;
}): Promise<QueryHistoryEntry[]> {
  const { userId, knowledgeBaseId, limit = 50 } = params;

  try {
    let query = supabase
      .from('rag_queries')
      .select(`
        id,
        query_text,
        mode,
        response_text,
        response_model,
        response_generation_time_ms,
        self_rag_score,
        self_rag_passed,
        citations,
        knowledge_base_id,
        created_at,
        rag_quality_scores (
          id,
          faithfulness,
          answer_relevance,
          context_relevance,
          answer_completeness,
          citation_accuracy,
          composite_score,
          evaluation_model,
          evaluation_time_ms,
          created_at
        )
      `)
      .eq('user_id', userId)
      .order('created_at', { ascending: false })
      .limit(limit);

    if (knowledgeBaseId) {
      query = query.eq('knowledge_base_id', knowledgeBaseId);
    }

    const { data: queries, error } = await query;

    if (error) {
      console.error('Error fetching query history:', error);
      throw new Error(`Failed to fetch query history: ${error.message}`);
    }

    if (!queries || queries.length === 0) {
      return [];
    }

    // Map to QueryHistoryEntry format
    const history: QueryHistoryEntry[] = queries.map((q: any) => {
      // rag_quality_scores is a 1:N relation but we expect 1:1 in practice
      const qualityScore = Array.isArray(q.rag_quality_scores)
        ? q.rag_quality_scores[0] || null
        : q.rag_quality_scores || null;

      return {
        queryId: q.id,
        queryText: q.query_text,
        mode: q.mode as RAGMode,
        responseText: q.response_text,
        responseModel: q.response_model,
        responseTimeMs: q.response_generation_time_ms,
        selfRagScore: q.self_rag_score ? Number(q.self_rag_score) : null,
        selfRagPassed: q.self_rag_passed,
        citations: q.citations,
        knowledgeBaseId: q.knowledge_base_id,
        createdAt: q.created_at,
        qualityScore: qualityScore
          ? {
              compositeScore: Number(qualityScore.composite_score),
              faithfulness: Number(qualityScore.faithfulness),
              answerRelevance: Number(qualityScore.answer_relevance),
              contextRelevance: Number(qualityScore.context_relevance),
              answerCompleteness: Number(qualityScore.answer_completeness),
              citationAccuracy: Number(qualityScore.citation_accuracy),
              evaluationModel: qualityScore.evaluation_model,
              evaluationTimeMs: qualityScore.evaluation_time_ms,
            }
          : null,
      };
    });

    return history;
  } catch (error) {
    console.error('Error in getQueryHistory:', error);
    throw error;
  }
}

/**
 * Get side-by-side quality comparison across the three modes.
 *
 * This is the **hero feature** data -- it quantitatively demonstrates that
 * RAG+LoRA > RAG alone > LoRA alone. The data powers the mode comparison
 * card on the quality dashboard.
 *
 * Groups quality scores by mode (rag_only, lora_only, rag_lora) and computes
 * averages per mode, enabling direct comparison.
 *
 * @param params - userId and knowledgeBaseId (required for meaningful comparison)
 * @returns Side-by-side comparison data for all three modes
 *
 * @example
 * ```typescript
 * const comparison = await getModeComparison({
 *   userId: 'user-123',
 *   knowledgeBaseId: 'kb-456',
 * });
 * // comparison.modes:
 * // {
 * //   rag_only:  { avgComposite: 0.78, queryCount: 15, ... },
 * //   lora_only: { avgComposite: 0.65, queryCount: 12, ... },
 * //   rag_lora:  { avgComposite: 0.88, queryCount: 20, ... },
 * // }
 * // comparison.bestMode: 'rag_lora'
 * // comparison.improvement: { ragLoraVsRagOnly: 12.8, ragLoraVsLoraOnly: 35.4 }
 * ```
 */
export async function getModeComparison(params: {
  userId: string;
  knowledgeBaseId: string;
}): Promise<ModeComparison> {
  const { userId, knowledgeBaseId } = params;

  try {
    // Fetch all quality scores for this knowledge base, joined with queries for mode
    const { data: scores, error } = await supabase
      .from('rag_quality_scores')
      .select(`
        faithfulness,
        answer_relevance,
        context_relevance,
        answer_completeness,
        citation_accuracy,
        composite_score,
        evaluation_model,
        rag_queries!inner (
          mode,
          knowledge_base_id
        )
      `)
      .eq('user_id', userId)
      .eq('rag_queries.knowledge_base_id', knowledgeBaseId)
      .neq('evaluation_model', 'error-fallback');

    if (error) {
      console.error('Error fetching mode comparison:', error);
      throw new Error(`Failed to fetch mode comparison: ${error.message}`);
    }

    if (!scores || scores.length === 0) {
      return buildEmptyModeComparison();
    }

    // Group by mode
    const byMode: Record<string, any[]> = {
      rag_only: [],
      lora_only: [],
      rag_lora: [],
    };

    for (const score of scores) {
      const queryData = score.rag_queries as any;
      const queryMode = queryData?.mode || 'rag_only';
      if (byMode[queryMode]) {
        byMode[queryMode].push(score);
      }
    }

    // Compute per-mode averages
    const modes: Record<string, ModeBreakdown> = {};
    for (const [modeKey, modeScores] of Object.entries(byMode)) {
      if (modeScores.length === 0) {
        modes[modeKey] = {
          avgComposite: 0,
          avgFaithfulness: 0,
          avgAnswerRelevance: 0,
          avgContextRelevance: 0,
          avgAnswerCompleteness: 0,
          avgCitationAccuracy: 0,
          queryCount: 0,
        };
      } else {
        modes[modeKey] = {
          avgComposite: average(modeScores.map(s => Number(s.composite_score))),
          avgFaithfulness: average(modeScores.map(s => Number(s.faithfulness))),
          avgAnswerRelevance: average(modeScores.map(s => Number(s.answer_relevance))),
          avgContextRelevance: average(modeScores.map(s => Number(s.context_relevance))),
          avgAnswerCompleteness: average(modeScores.map(s => Number(s.answer_completeness))),
          avgCitationAccuracy: average(modeScores.map(s => Number(s.citation_accuracy))),
          queryCount: modeScores.length,
        };
      }
    }

    // Determine best mode
    const modeEntries = Object.entries(modes).filter(([_, m]) => m.queryCount > 0);
    const bestMode = modeEntries.length > 0
      ? modeEntries.reduce((best, current) =>
          current[1].avgComposite > best[1].avgComposite ? current : best
        )[0] as RAGMode
      : null;

    // Calculate improvement percentages (RAG+LoRA vs others)
    const ragLoraAvg = modes.rag_lora?.avgComposite || 0;
    const ragOnlyAvg = modes.rag_only?.avgComposite || 0;
    const loraOnlyAvg = modes.lora_only?.avgComposite || 0;

    const improvement = {
      ragLoraVsRagOnly: ragOnlyAvg > 0
        ? Number(((ragLoraAvg - ragOnlyAvg) / ragOnlyAvg * 100).toFixed(1))
        : 0,
      ragLoraVsLoraOnly: loraOnlyAvg > 0
        ? Number(((ragLoraAvg - loraOnlyAvg) / loraOnlyAvg * 100).toFixed(1))
        : 0,
    };

    const comparison: ModeComparison = {
      modes,
      bestMode,
      improvement,
      totalQueries: scores.length,
    };

    return comparison;
  } catch (error) {
    console.error('Error in getModeComparison:', error);
    throw error;
  }
}

/**
 * Compute arithmetic mean of an array of numbers.
 * Returns 0 for empty arrays.
 */
function average(values: number[]): number {
  if (values.length === 0) return 0;
  const sum = values.reduce((acc, val) => acc + val, 0);
  return Number((sum / values.length).toFixed(2));
}

/**

/**
 * Compute daily trend data for charting (quality over time).
 * Groups scores by date and computes daily averages.
 */
function computeTrendData(
  scores: any[]
): Array<{ date: string; avgComposite: number; queryCount: number }> {
  const dailyMap = new Map<string, { compositeSum: number; count: number }>();

  for (const score of scores) {
    const date = score.created_at?.split('T')[0];
    if (!date) continue;

    if (!dailyMap.has(date)) {
      dailyMap.set(date, { compositeSum: 0, count: 0 });
    }

    const day = dailyMap.get(date)!;
    day.compositeSum += Number(score.composite_score);
    day.count += 1;
  }

  const trendData = Array.from(dailyMap.entries())
    .map(([date, { compositeSum, count }]) => ({
      date,
      avgComposite: Number((compositeSum / count).toFixed(2)),
      queryCount: count,
    }))
    .sort((a, b) => a.date.localeCompare(b.date));

  return trendData;
}

11. `getModeComparison` groups scores by mode and computes per-mode averages
12. `getModeComparison` identifies the best-performing mode and computes percentage improvement of RAG+LoRA vs each other mode
13. All functions handle empty result sets gracefully (return zero-value objects, not errors)
14. All scores are rounded to 2 decimal places and clamped to [0, 1]
